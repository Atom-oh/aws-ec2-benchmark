종합 벤치마크 대시보드

11개 벤치마크의 대표 지표를 54개 EC2 인스턴스 타입 기준으로 통합 비교

리전: ap-northeast-2 (서울) | 인스턴스 크기: xlarge (4 vCPU)

목차

정규화 방법론

각 벤치마크는 자체 headline 지표와 방향성을 제공한다. 값이 클수록 좋은 지표는 value / best * 100으로, 값이 작을수록 좋은 지표는 best / value * 100으로 변환해 벤치마크별 최고 성능을 100점으로 맞춘다.

벤치마크별 히트맵

종합 랭킹

결론

종합 탭은 벤치마크별 대표 지표를 동일한 0-100 점수 체계로 맞춰 비교의 기준을 제공한다. 세부 워크로드 판단은 각 벤치마크 탭의 원본 지표와 함께 확인한다.
