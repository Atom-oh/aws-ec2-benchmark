테스트 방법론

Spring Boot PetClinic 애플리케이션에 대해 wrk 부하를 50, 100, 200 동시 연결 단계로 나누어 측정했다. 각 단계는 동일한 애플리케이션 엔드포인트를 대상으로 초당 요청 수와 지연 시간을 기록하며, 부하가 높아질 때 처리량이 얼마나 유지되는지 비교한다. 콜드스타트는 애플리케이션 프로세스 시작 후 서비스가 HTTP 요청을 정상 처리할 수 있을 때까지의 시간을 초 단위로 측정했다.
